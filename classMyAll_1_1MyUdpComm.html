Detailed Description

My class for communicating with UDP.

This creats a object to comunicate with partners using UDP connections. Please call Init() right after creating the object. If one partner is selected in "Init()", you can communicate with only the partner using "Recv_partner()", while "MyUdpMaster::Recv()" will accept any messages. If you wanna use like broadcast/multicast, you shoud set an appropriate partner's IP address. For example,

  • "255.255.255.255" : limited broadcast address
  • "192.168.1.255" : directed broadcast address. This communicates with under the 192.168.1.*
  • "224.0.0.0" to "224.0.0.255" : multicast address (link-local scope)
  • "224.0.1.0" to "238.255.255.255" : multicast address (global scope)
  • "239.0.0.0" to "239.255.255.255" : multicast address (private scope)
