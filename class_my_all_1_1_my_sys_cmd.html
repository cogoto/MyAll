Detailed Description

My Class for ordering system commands.

This class object could work as belows,

  • Execute system commands and receive its replies.
  • Honk beep sounds.
  • Print standard output with front & back colors, bold-character & underscore. Some environments might need "gnu++11" to compile this class.
