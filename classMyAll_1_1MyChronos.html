Converts the format. :

This function converts a real number to a "day/hour/minute/second" unit.

Parameters
[in]date__: A real number.
[in]date_unit__: The unit of the real number :
  • day : ex. date__ = 12.5 -> 12 day 12:00:00.0
  • hour : ex. date__ = 10.5 -> 0 day 10:30:00.0
  • min : ex. date__ = 55.5 -> 0 day 00:55:30.0
  • sec : ex. date__ = 86411.1 -> 1 day 00:00:11.1
Returns
The date array.
  • [0] : Day
  • [1] : Hour
  • [2] : Minute
  • [3] : Second
  • [4] : The decimal part of the second [second] If failed, this would return {-1}.
