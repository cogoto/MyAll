Detailed Description

My class for managing status.

This class object could manage normal/error statuses and status values. This class can do the followings:

  • Performing the management of command statement.
  • Performing the management of binary command.
  • Performing the management of macro command statement.
  • Performing the status management.
  • Performing the status-value management.

◆ Connect()

void MyAll::MyStatus::Connect ( const std::string &  sts_name__,
const std::string &  val_name__,
const std::string &  match_val_1__,
const std::string &  match_val_2__ = "" 
)

Associates the status with the status-value. :

This function associates the status with the status-value. If the result evaluated by matching-patterns is true, the associated status becomes true. The result from two matching-patterns is logical AND, basically. However, the below patterns are evaluated by logical OR.

  • "< MIN", "> MAX" : A < MIN or A > MAX
  • "<= MIN", "> MAX" : A <= MIN or A > MAX
  • "< MIN", ">= MAX" : A < MIN or A >= MAX
  • "<= MIN", ">= MAX" : A <= MIN or A >= MAX Here, the 1st & 2nd pattern is correcpond to val_match_1__ & val_match_2__, A is the status-value and MIN <= MAX.
    Parameters
    [in]sts_name__: The status name.
    [in]val_name__: The status-value name.
    [in]val_match_1__: The matching-pattern to evaluate the status value.
    The pattern examples are as below,
  • "== A", "!= A"
  • "> A", ">= A", "< A", "<= A"
  • "include A (inc A)", "exclude A (exc A)". Here, A is a status-value.
    Parameters
    [in]val_match_2__: The matching-pattern to evaluate the status value. Please see the above.
    Returns
    None.
