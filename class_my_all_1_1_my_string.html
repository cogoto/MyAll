Detailed Description

My class for dealing string.

This class operates string as below,

  • Splitting string with regular explessions.
  • Replacing string with regular explessions.
  • String(s) to value(s) etc...

String splitting with a delimiter. :

This function splits a string with a delimiter.

Parameters
[out]result__: A splited string array.
[in]text__: A string you wanna split.
[in]delimiter__: A delimiting Pattern. The default delimiter is blank. The pattern is described by the regular expression (ECMAScript), the examples are as belows,
  • R"(\s+)" ... Splits by blanks.
  • R"(\d+)" ... Splits by integers ([0-9]+) only.
  • R"(\w+)" ... Splits by words.
  • R"(abc)" ... Splits by "abc".
  • R"(\+)" ... Splits by "+" character.
Returns
None.
